[Final Master PRD] 위플마켓: Anti-Gravity Edition
1. Project Overview
프로젝트명: 위플마켓 (Wiple Market) - Anti-Gravity Platform

서비스 정의: 마케팅의 물리적 제약(약정/비용/위약금)을 제거한 모듈형 비즈니스 솔루션.

Design Concept: Anti-Gravity (무중력), Glassmorphism, Deep Space UI.

Core Message: "Gravity Off. Growth On. (무게는 끄고, 성장을 켭니다.)"

Target Audience: 비효율적인 관행을 거부하고 합리적인 성과를 추구하는 스마트 비즈니스 오너.

2. Visual & Design Strategy
Metaphor:

Old Marketing = Gravity (중력): 1년 약정, 위약금, 고정비 등 비즈니스를 짓누르는 요소.

Wiple Market = Space Station (우주 정거장): 필요한 부품(모듈)만 도킹하여 추진력을 얻는 플랫폼.

Visual Key:

Floating Object: 스크롤 시 텍스트와 오브젝트가 무중력 상태처럼 부유하는 인터랙션.

Glass Panel: 배경이 비치는 반투명한 UI 패널로 미래지향적 느낌 구현.

Neon Accent: 어두운 배경(#0B0C15) 위에서 빛나는 Cyan(#00F0FF) 및 Indigo(#6366F1) 컬러.

4. Functional Specifications (기능 명세)
4.1 Frontend Tech Stack
Core: React.js or Next.js (SEO 최적화).

Animation:

GSAP (GreenSock): 스크롤 기반의 복합적인 인터랙션 구현 (Floating, Parallax).

Three.js (Optional): Hero 섹션의 3D 오브젝트(바위 분해) 구현 시 사용. (성능 고려하여 영상이나 Lottie로 대체 가능).

Styling: Tailwind CSS (Glassmorphism 유틸리티 클래스 커스텀).

4.2 Module Cart Logic (장바구니)
Interaction: '도킹(담기)' 버튼 클릭 시, 해당 모듈 아이콘이 궤적을 그리며 하단 바로 이동하는 애니메이션.

State Management:

선택된 모듈의 ID, Price, Option을 전역 상태(Zustand/Redux)로 관리.

하단 바에서 실시간 합계 금액 계산 (천 단위 콤마 자동 포맷팅).

Submit: 폼 제출 시 선택된 모듈 데이터 배열(Array)을 함께 전송.

Layout: 카드 UI는 모바일에서 가로 스크롤(Swipe) 형태로 변환하여 공간 효율성 확보.
